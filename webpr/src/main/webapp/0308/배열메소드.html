배열 요소의 위치 찾기
indexOf() : 맨 앞에서부터 먼저 같은 요소의 index값 반환
lastIndexOf() : 맨 뒤에서부터 같은 요소의 index값 반환
중간에 있는 것을 찾고 싶으면 start위치를 지정해서 찾아야함.





배열요소의 추가/삭제
push(a,b,c,...) : 마지막에 요소를 추가
pop() : 마지막 요소를 제거하여 리턴





배열 요소의 위치 찾기

lastIndexOf() : 맨 뒤에서부터 같은 요소의 index값 반환

배열요소의 추출
slice(a,b,c,...) : start ~ end-1 까지 추출해서 새로운 배열을 만든다.





배열의 수정 - 삽입, 삭제
splice(index, n, a,b,c,...) : index위치에서 n개를 삭제하고, a, b, c,... 아이템을 삽입한다.
n이 0이면 삭제없이 아이템을 추가한다. splice(index, n) : index위치에서 n개를 삭제한다.





배열의 문자열 변환
join(구분자) : 구분자를 지정한다. 구분자를 생략하면 ,를 기준으로 한다.
arr=["사과", "복숭아", "참외", "사과", "파인애플", "바나나", "사과", "망고"];
join 문자열 = "사과,복숭아,참외,사과,파인애플,바나나,사과,망고"로 바뀜
